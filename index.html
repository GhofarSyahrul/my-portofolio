skills

HTML

Sebagai seorang pengembang web, saya memiliki keahlian yang kuat dalam HTML, bahasa dasar yang mendasari struktur setiap halaman web. Saya mampu membuat markup yang semantik dan bersih, memastikan kompatibilitas lintas perangkat, serta mengoptimalkan aksesibilitas untuk pengguna.

CSS

Keterampilan saya dalam CSS mencakup kemampuan untuk mendesain tata letak yang menarik dan responsif. Saya dapat mengimplementasikan desain visual dengan presisi, menggunakan konsep-konsep seperti flexbox dan grid untuk mencapai tata letak yang dinamis.

Adobe Photoshop

Keahlian saya Adobe Photoshop mencakup pengeditan gambar berupa poster. Saya mampu mengoptimalkan gambar untuk berbagai platform dan memastikan kualitas visual yang konsisten. Dengan pemahaman yang kuat tentang lapisan, mask, dan alat-alat khusus Photoshop, saya dapat mencapai efek visual yang diinginkan sesuai dengan kebutuhan proyek.

adobe illustrator

Saya juga terampil dalam mengelola warna, gradient, dan efek untuk memberikan sentuhan pada desain.

FIGMA

Pemahaman saya tentang desain responsif memastikan bahwa setiap elemen dapat diakses dan dinikmati oleh pengguna pada berbagai perangkat.
